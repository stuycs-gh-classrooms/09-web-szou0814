My modified version of the Koch Curve is meant to be a snowflake and includes the following modificaitons:

  • As depth decreases, the color of each depth 1 line randomly changes into one of seven options.
  • Woah! Our snowflake is crystallizing! As depth decreases, two counters will increase by 1 and 0.05, respectively, for every depth. The initial given depth represents how large our snowflake is. If the snowflake has a depth of larger than 3, the first counter (c) will be greater than or equal to 2 and the snowflake will start to form crystals by stamping a turtle after every depth 1 line. The greater the initial given depth, the greater the second counter (s) and larger the crystals will be.
  • Yipee! Our snowflake is growing! As depth decreases, the length and width will increase by 1.35x and 1.2x, respectively, for every depth. The initial given depth determines how large our snowflake is as it determines how many times the length and width will be multiplied.

def pen_color(t):
    x = random.randrange(0, 7)
    if x == 0:
        t.color('aquamarine')
    elif x == 1:
        t.color('aquamarine3')
    elif x == 2:
        t.color('cadetblue1')
    elif x == 3:
        t.color('cadetblue3')
    elif x == 4:
        t.color('lightblue1')
    elif x == 5:
        t.color('lightblue3')
    else:
        t.color('lightskyblue')
        
def crystals(t, c, s):
    if c >= 2:
        t.shape('arrow')
        t.turtlesize(0.5 + s)
        t.stamp()
        t.shape('classic')
        t.turtlesize(1)
        
def draw_koch_modified(t, depth, length, width, c, s):
    pen_color(t)
    t.width(width)
    if depth == 1:
        t.fd(length)
        crystals(t, c, s)
    if depth > 1:
        draw_koch_modified(t, depth - 1, length * 1.35, width * 1.2, c + 1, s + 0.05)
        t.lt(60)
        draw_koch_modified(t, depth - 1, length * 1.35, width * 1.2, c + 1, s + 0.05)
        t.rt(120)
        draw_koch_modified(t, depth - 1, length * 1.35, width * 1.2, c + 1, s + 0.05)
        t.lt(60)
        draw_koch_modified(t, depth - 1, length * 1.35, width * 1.2, c + 1, s + 0.05)
        
def draw_koch_snowflake(t, depth, length, width, c, s):
    i = 0
    
    while i < 3:
        draw_koch_modified(t, depth, length, width, c, s)
        t.rt(120)
        
        i += 1

My modified version of the Tree is meant to be a tree representing the four seasons and includes the following modifications:

  • Oh, the seasons! As depth decreases, the color of the leaves randomly changes to represent each of the four seasons. When representing summer, the leaves are green. When representing spring, the leaves are pink, similar to cherry blossoms. When representing autumn, the leaves have a random, 50% chance of being orange or brown, similar to the falling autumn leaves. When representing winter, the leaves are brown, similar to a barren tree branch. The branches are always brown.
  • Watch out! Whenever depth decreases, lightning has a 1% success rate of striking tree branch, causing it to not be drawn. The greater the initial given depth, the greater the overall amount of branches that will likely be striken down due to the lightning being given more chances.
  • It's just the natural way of life! As depth decreases, the width of the branches and the angle at which the branches are grown randomly changes to a number within a given range.

def seasons(t):
    x = random.randrange(0, 4)
    if x == 0:
        t.color('green')
    elif x == 1:
        t.color('hotpink')
    elif x == 2:
        y = random.randrange(0, 2)
        if y == 0:
            t.color('brown')
        else:
            t.color('orangered')
    else:
        t.color('brown')

def tree_modified(t, depth, length):
    t.width(random.randrange(1, 11))
    angle = random.randrange(10, 45)
    if depth == 1:
        seasons(t)
        t.fd(length)
        t.bk(length)
        t.color("brown")

    else:
        t.color("brown")
        t.fd(length)
        t.rt(angle)
        a = random.randrange(0, 100)
        if a != 0:
            tree_modified(t, depth - 1, length * .75)
        t.lt(angle * 2)
        a = random.randrange(0, 100)
        if a != 0:
            tree_modified(t, depth - 1, length * .75)
        t.rt(angle)
        t.bk(length)

My modified version of the Sierpinski Triangle is meant to be a star and includes the following modifications:

  • Even the brightest starts eventually burn out! As depth decreases, the RGB value of green increases, changing the color of the star from bright, deep red to lighter oranges. When the RGB value of green reaches 255, the star will begin to burn out and dim as it entirely becomes a light yellow. The initial given depth represents how old our star is. If the star has a greater depth, the greater the RGB value of green will be due to it being added to greater times. Each sierpinski triangle of one less depth that is drawn will sequentially add a greater value to the RGB value of green, providing a gradient effect.
  • It's the end of the world (literally)! As depth decreases, there is a 0.25% chance that a black hole will swallow the star and cause the turtle world to close. The greater the initial given depth, the greater the overall likelihood that a black hole will swallow the sun due to the black hole being given more chances.

def triangle(t, size):
    t.lt(60)
    t.fd(size)
    t.rt(120)
    t.fd(size)
    t.rt(120)
    t.fd(size)
    t.rt(180)

def draw_sierpinski_modified(t, depth, length, width, g):
    t.width(width)
    
    if g < 255:
        t.color(255, g, 0)
    else:
        t.color('yellow')

    if depth == 1:
        triangle(t, length)
    if depth > 1:
        draw_sierpinski_modified(t, depth - 1, length / 2, width, g + 10)
        t.fd(length / 2)
        draw_sierpinski_modified(t, depth - 1, length / 2, width, g + 40)
        t.bk(length / 2)
        t.lt(60)
        t.fd(length / 2)
        t.rt(60)
        draw_sierpinski_modified(t, depth - 1, length / 2, width, g + 60)
        t.rt(120)
        t.fd(length / 2)
        t.lt(120)
        
    x = random.randrange(0, 400)
    if x == 0:
        t.shape('circle')
        t.turtlesize(length)
        t.color('black')
        t.stamp()
        turtle.bye()

def draw_star(t, depth, length, width, g):
    draw_sierpinski_modified(t, depth, length, width, g)
    t.fd(length)
    t.lt(180)
